# 🚀 ISRO Weather & Satellite Information Web App

A Django-based web application that displays ISRO satellite information along with real-time weather data using API integration. This project demonstrates backend development, API handling, deployment, and debugging skills.

---

## 🌍 Live Demo
🔗 https://isro-weather-app.onrender.com/

---

## 📌 Features

- 🛰 ISRO satellite information display
- 🌦 Real-time weather data using API
- 🖥 Django backend implementation
- 🔗 API integration using Python requests
- ☁ Cloud deployment using Render
- 🛠 Production debugging experience
